# e-plantShopping

## Paradise Nursery

e-plantShopping is a React and Redux based shopping cart application for Paradise Nursery.

The application allows users to:

* Browse indoor plants grouped by categories
* View plant details including image, name, description and price
* Add plants to the shopping cart
* Increase or decrease item quantities
* Remove items from the cart
* View the total cost of all items
* Continue shopping or proceed to checkout

This project was developed as part of the IBM Front-End Development React Final Project.
